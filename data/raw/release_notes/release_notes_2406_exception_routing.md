# Release 2406 Exception Routing Improvements

## Overview
Release 2406 improved how exception cases are routed to Revenue Operations.

## What changed
Protected exception flags now trigger earlier routing to Revenue Operations.
Strategic renewal candidates are identified sooner in the workflow.

## Support impact
Support may notice that some cases previously handled locally are now routed out automatically.
This does not mean the workflow failed; it may mean stricter classification is working as intended.

## Notes
Support should collect better deal metadata before attempting any remediation.
