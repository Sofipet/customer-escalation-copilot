# Release 2407 Renewal Threshold Update

## Overview
Release 2407 updated threshold handling for renewals with retention risk.

## What changed
Renewal deals with contract value above €100,000 now trigger stricter approval routing even at moderate discount levels.
Some support users may see renewals routed to director review more often than before.

## Support impact
Support should verify whether a case is classified as a renewal before relying on standard-deal guidance.
Local manual override habits may no longer apply in borderline renewal scenarios.

## Notes
This release increased the importance of correct deal classification.
