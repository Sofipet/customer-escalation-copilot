# Release 2402 Quote Rules Update

## Overview
Release 2402 aligned quote approval routing for standard deals with the updated threshold service.

## What changed
Standard deals with discounts below 10% now use a more stable automatic approval path.
Support was advised to monitor false failures during the first two weeks after rollout.

## Support impact
Support may continue to use established troubleshooting steps for routing timeouts.
No change was made to strategic renewal behavior in this release.

## Notes
This release established the baseline for later discount and routing updates.
